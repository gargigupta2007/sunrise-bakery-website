# Sunrise Bakery Website Project

This folder contains a complete, responsive website for a real-business style assignment.

## Pages Included
- `index.html` (Homepage)
- `about.html` (About Us)
- `services.html` (Products/Services)
- `contact.html` (Contact + Inquiry Form)

## Features
- Mobile-friendly responsive design
- Sticky navigation with mobile menu toggle
- Product/service cards
- Testimonials section
- Inquiry form with frontend validation

## Customization Steps
1. Replace `Sunrise Bakery` content with your selected business details.
2. Update address, phone, email, and business hours.
3. Replace sample product names/prices.
4. Add real images under `assets/images` and reference them in pages.

## Publish to GitHub Pages
1. Create a GitHub repository and upload all files from `website`.
2. Open repository `Settings` > `Pages`.
3. Set source to `Deploy from a branch`.
4. Select branch `main` and folder `/root`.
5. Save and wait 1-2 minutes for your live link.

## Submission Requirement

- GitHub repository link
